--- picture: "/assets/images/KO.png" name: "Realtime Multiplayer Knockout" link: "https://aloinpenguins.herokuapp.com/" --- This is an online multiplayer version of 'knockout' made using Javascript with networking features of Node.JS and Socket.io It also features realistic ball collision physics all done using pure JS calculations. Network capabilities include unique names, room creation, room joining, and a lobby of up to 8 players.
